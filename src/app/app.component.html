This calculator supports Decimal, Hexadecimal, and Binary number inputs. press Enter to push the current value to the stack. Escape to pop the top value off the stack.

Use the mode buttons or the left/right arrow buttons to switch between modes.

You can perform calculations using standard operators (+, -, *, /) and view the results in all three number systems simultaneously.

Boolean operations such as AND (&), OR (|), XOR (^), and NOT (!) are also supported.

This calculator was created to assist programmers in converting and calculating values across differe nt number systems commonly used in programming. It aims to provide a user-friendly interface for quick conversions and calculations.

Being a Programmer's Calculator, it is assumed you understand how to run Reverse Polish Notation (RPN) calculations; the operation (e.g. *) runs on the top two stack values.

I wrote the original version of this calculator in Classic C, around 1985 during a lull in production demands. What took days to code then, now takes hours. This Angular version is a modern reimagining of that original program, updated to leverage contemporary web technologies while preserving the core.
